MR. HAPPY
Released in June 2021, Mr. Happy is my first published game. This game went through a tumultuous development process, first being canned all the way back in 2019 because the code had gotten so spaghettified that I could not figure it out anymore (I'm not kidding when I say that). I then came up with other new (and completely overambitious) approaches to Mr. Happy before picking it back up again in 2021 and sticking with the original idea.
